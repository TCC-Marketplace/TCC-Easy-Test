⚙️ GAS Admin Actions

            

              Admin สามารถเรียก GAS actions เพิ่มเติมได้จากหน้านี้ ระบบจะส่ง request ไปยัง SCRIPT_URL ของ TCC
            

            
getAllUsers
ดึงข้อมูล Users sheet ทั้งหมด

            
getDeposits
ดึงรายการ logDeposit

            
getWithdraws
ดึงรายการ logWithdraw

            
getOpenPositions
ดึง positions Status=Active

            
getAllEarnContracts
ดึง earnContracts ทุก user

            
⚠️ Actions เหล่านี้ต้องเพิ่มใน GAS script ก่อนใช้งาน ดูไฟล์ tcc_gas_admin.gs
